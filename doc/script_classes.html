loadfile( filename )Load bitmap data from a file.
getSubBitmap( x, y, w, h )Returns a sub-bitmap with the given co-ordinates and width and height.
setBitmap( Bitmap )Replace bitmap's current internal ALLEGRO_BITMAP* with one from given Bitmap.
